# AGENTS.md

## Regras obrigatórias de validação

Antes de concluir qualquer tarefa neste repositório, sempre execute:

1. `npm run format`
2. `npm run check`

## Qualidade

- Nunca ignore falhas de lint, prettier, testes ou build.
- Nunca desabilite testes apenas para fazer a suíte passar.
- Corrija a causa real do problema.
- Mantenha mudanças pequenas, seguras e revisáveis.

## Finalização

Ao concluir uma tarefa, informe:

- arquivos alterados
- comandos executados
- resultado de cada validação
- riscos remanescentes, se houver
