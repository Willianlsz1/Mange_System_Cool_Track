
# CoolTrack Pro 🧊

**Sistema de Gestão de Manutenção de Equipamentos de Ar Condicionado**

CoolTrack Pro é uma aplicação web moderna para gerenciamento completo de manutenção de sistemas HVAC (Heating, Ventilation, and Air Conditioning), desenvolvida com design industrial dark mode e interface mobile-first.

---

## ✨ Funcionalidades

- 📊 **Dashboard Interativo** - Visão geral do parque de equipamentos com KPIs e health scores
- ❄️ **Gestão de Equipamentos** - Cadastro completo com TAG, localização, tipo e fluido refrigerante
- 📝 **Registro de Manutenções** - Controle detalhado de serviços preventivos e corretivos
- 📅 **Histórico Timeline** - Visualização cronológica de todas as manutenções
- 🔔 **Sistema de Alertas** - Notificações automáticas para manutenções vencidas e equipamentos críticos
- 📄 **Relatórios** - Geração de relatórios por período com opção de impressão
- 📷 **Anexo de Fotos** - Compressão automática e lightbox para visualização
- 📱 **Responsivo** - Interface adaptável para desktop, tablet e mobile
- 🌙 **Dark Mode** - Design industrial com alto contraste

---

## 🚀 Tecnologias

- **HTML5** & **CSS3** (Custom Properties, Grid, Flexbox)
- **JavaScript ES6+** (Modules, Async/Await)
- **Vite** ⚡ - Build tool e dev server ultrarrápido
- **Google Fonts** - Inter (UI) + JetBrains Mono (código)

---

## 📋 Pré-requisitos

- **Node.js** 16.20+ ou 18.12+ (recomendado: LTS mais recente)
- **npm** 7+ (ou yarn/pnpm equivalentes)
- Navegador moderno (Chrome, Firefox, Edge, Safari)
